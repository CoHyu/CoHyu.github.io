外观配置

Oh-My-Posh（提示符美化）

PowerShell 直接安装 + 获取主题预览：

  • Install-Module oh-my-posh -Scope CurrentUser
  • Get-PoshThemes

我这边命令缺失, 无法直接查看, 给出官网:

  • https://ohmyposh.dev/docs/themes
  • 然后选择一个你喜欢的款式, 记下名字
  • 然后在 $profile 添加内容 ( 持久化加载 ): oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH/clean-detailed(你选择的那款的名字).omp.json" | Invoke-Expression

PS: 个人测试, 有的主题就是加载速度很慢, 值得注意一下

Font（字体）

去 iterm-2-colors-chemes ( 一个开源项目 ) 找配色

先在 Readme.md 里选一个喜欢的颜色，比如 Aurora，然后去”https://github.com/mbadolato/iTerm2-Color-Schemes/blob/master/windowsterminal/Aurora（你选择的颜色名）.json“ 这里，将里面的内容添加到 Windows Terminal 的 json 配置文件里的 “schemes” 中，然后在终端设置一下主题就行

终端的 JSON 在打开终端设置的左下角
